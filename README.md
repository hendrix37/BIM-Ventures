## Deployment

To clone project

```bash
  git clone https://github.com/hendrix37/BIM-Ventures.git
```

set .env use local database

```bash
  cp .env.example .env
```

set migration an create dummy data

```bash
  php artisan migrate --seed
```

run project

```bash
  php artisan serve
```
